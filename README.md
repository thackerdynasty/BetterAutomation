# BetterAutomation
## What is it?
BetterAutomation is a 1.21.1 Minecraft mod that enhances the automation capabilities of the game. It introduces new blocks, items, and mechanics that allow players to create more complex and efficient automated systems.
## Features
- New automation blocks: Conveyor belts, robotic arms, and more.
- Advanced item sorting and storage solutions.
- Integration with existing mods for expanded functionality.
- Customizable automation recipes and processes.
## Installation
Currently, as the mod is in pre-alpha, there are no .jar files available for download. To use the mod, you will need to clone the repository and build it from source.
